Learning Blender

Old monkey need new tricks and will note and share for future self reference.

Tips

  • Down below there is a small bar the will help remember shortcuts
  • Left bar is usually dependent on which mode is selected and has the tools for it
  • Top left is mode menu
  • Top right is gizmos menu (zoom and etc) and above it a view mode menu (x-ray, gizmos hiding, etc)
  • In Ubuntu, Context Menu is Right Click

Modifiers

  • Subdivision Surface - creates more divisions automatically. Good to round corners
  • Solidify - Helps giving and adjusting thickness
  • Array modifiers - Allows to add the mesh multiple times

Editing

  • There is the proportional editing on top right menu witch uses proportional weights to execute modifications

Shortcuts

  • Tab - Switch between Edit and Object Modes
  • G - Grab
  • S - Scale
  • R - Rotate
  • Shift + S - Open menu to Change cursor positions
  • Numpad 1,3,5… Change viewport view position
